# Análise de Dados com Python

Este repositório reúne os notebooks desenvolvidos durante o curso **Python para Dados** da Asimov Academy. O projeto tem como objetivo aplicar, na prática, conceitos de programação e análise de dados utilizando Python.

## Sobre o curso

As aulas utilizam dados reais do Portal da Transparência do Brasil, permitindo uma experiência prática na organização, tratamento e visualização de informações.

## Objetivo deste repositório

Este repositório foi criado para:

* Consolidar os conhecimentos adquiridos ao longo do curso
* Documentar a evolução no aprendizado de análise de dados
* Servir como portfólio de projetos em Python

## Estrutura do projeto

* `analise.ipynb`: notebook final com a análise completa dos dados
* `teste.ipynb`: notebook utilizado para experimentações e testes durante o aprendizado

## Principais habilidades desenvolvidas

* Fundamentos de programação com Python
* Manipulação de dados com Pandas
* Limpeza e tratamento de dados
* Aplicação de filtros e agrupamentos
* Criação de gráficos e visualizações
* Uso do Google Colab para análise de dados

## Ferramentas e tecnologias

* Python
* Pandas
* Google Colab
* Matplotlib

## Fonte dos dados

Os dados utilizados nas análises são provenientes do Portal da Transparência do Brasil, utilizados para fins educacionais.
